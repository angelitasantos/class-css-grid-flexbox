### Alinhamento

justify = **eixo x** 
posicionamento horizontal, da esquerda para a direita

align = **eixo y**
posicionamento vertical, de cima para baixo


1. `justify-content`
2. `align-content`
3. `justify-itens`
4. `align-itens`
5. `justify-self`
6. `align-self`

    - row-reverse
    - column-reverse

    - wrap
    - wrap-reverse

    - flex-start
    - flex-end
    - space-between
    - space-around
- align-items: 
    - flex-start
    - flex-end
    - center
    - stretch
